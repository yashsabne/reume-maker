## 🛠 Setup & Installation  

1. **Clone the repository**  
   ```sh
   git clone https://github.com/your-repo/kashvi-saree.git
   cd kashvi-saree
   ```

2. **Install dependencies**  
   ```sh
   npm install
   ```

3. **Run backend**  
   ```sh
   cd server
   node index.js
   ```

4. **Run frontend**  
   ```sh
   cd client
   npm start
   ```

5. **Open the project**  
   Navigate to `http://localhost:3000/` in your browser.
